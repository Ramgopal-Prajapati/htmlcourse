HTML Tags Explained

üèóÔ∏è Document Structure Tags

üìÑ <html>

The root element of an HTML document. All other elements are contained within this tag.

Use Case Wrapping the entire HTML document content. 
<html lang="en">
  <head>...</head>
  <body>...</body>
</html>
üë§ <head>

Contains meta-information about the document, like title, links to stylesheets, and scripts.

Use Case Adding metadata, linking CSS, setting the page title. 
<head>
  <title>My Page</title>
  <meta charset="UTF-8">
</head>
üìù <body>

Contains all the visible content of the HTML document that users see in the browser.

Use Case Adding headers, paragraphs, images, and other visible elements. 
<body>
  <h1>Welcome</h1>
  <p>This is my page.</p>
</body>
üè∑Ô∏è <title>

Defines the title of the document shown in browser's title bar or tab.

Use Case Setting page title for browsers and search engines. 
<title>My Webpage - Home</title>
üîß <meta>

Provides metadata about the HTML document such as character set, viewport settings, and keywords.

Use Case Setting character encoding, viewport for responsive design, SEO descriptions. 
<meta charset="UTF-8">
<meta name="viewport" 
      content="width=device-width, initial-scale=1.0">

üìù Text Formatting Tags

# <h1>-<h6>

Heading tags define headings, with h1 being the most important and h6 the least.

Use Case Creating document structure with hierarchical headings. 
<h1>Main Title</h1>
<h2>Section Title</h2>
<h3>Subsection Title</h3>
üìÑ <p>

Defines a paragraph of text. Browsers add margin before and after paragraphs.

Use Case Writing articles, descriptions, or any block of text. 
<p>This is a paragraph of text that 
contains multiple sentences.</p>
üîó <a>

Creates hyperlinks to other web pages, files, locations, or email addresses.

Use Case Navigation between pages, external references, download links. 
<a href="https://example.com"
   target="_blank">
  Visit Example
</a>
üí™ <strong>

Indicates that its contents have strong importance, seriousness, or urgency.

Use Case Emphasizing important text with semantic meaning. 
This is <strong>very important</strong> text.
‚úçÔ∏è <em>

Marks text that has stress emphasis. Browsers typically display as italic.

Use Case Adding emphasis to words or phrases in sentences. 
I <em>really</em> want to go.
üî§ <span>

Generic inline container for phrasing content, used to style parts of text.

Use Case Styling specific parts of text without line breaks. 
This is <span style="color: red;">red text</span>.
üì¶ <div>

Generic container for flow content with no semantic meaning.

Use Case Grouping elements for styling or scripting purposes. 
<div class="container">
  <p>Content here</p>
</div>
‚Ü©Ô∏è <br>

Produces a line break in text. Useful for addresses or poems.

Use Case Adding line breaks within text without starting new paragraphs. 
First line<br>
Second line<br>
Third line
‚ûñ <hr>

Represents a thematic break between paragraph-level elements.

Use Case Separating content sections visually with a horizontal line.

<hr>
<p>Second section</p>
